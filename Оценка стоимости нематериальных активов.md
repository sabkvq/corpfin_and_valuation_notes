Для нематериальных активов нет постпрогнозного периода, потому что у патента есть конкретный срок "жизни". А для ЭВМ смотрим на моральное устаревание технологии.

Для стоимости активов используем [[Чистая приведенная стоимость|NPV]] (с -I и суммой потоков по периодам)
В качестве затрат:
- [[Первичные бухгалтерские документы]] (оплата патента)
- Вознаграждение изобретателя

Прогнозирование денежного потока, примеры:
- Уменьшение брака (разница затрат без и с изобретением)

>Ставка дисконтирования для изобретения:
>Можно просто использовать таблицу рекомендуемых значений (методичка РОСНАНО от Deloitte)
