Приведенная к сегодняшнему моменту
X -- деньги в будущем
$$PV(X \text{ at } t=n) = \frac{X}{(1+r_f)^n}$$

Discrete period compounding: $PV = 1/(1 + r)^T$
Continuous compounding: $PV = e^{−r_{cc}T}$

[[Дисконтирование|Discounting]]

Для денежного потока:
$$PV=\sum\frac{CF_t}{(1+r)^t}$$


используется в [[Чистая приведенная стоимость|NPV]]
