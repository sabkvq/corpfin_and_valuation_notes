Требования: ПБУ 4/99


1. Положение по бухгалтерскому учету «Учетная политика организации» (ПБУ 1/2008).

2. Положение по бухгалтерскому учету «Доходы организации» (ПБУ 9/99).

3. Положение по бухгалтерскому учету «Расходы организации» (ПБУ 10/99).

4. Положение по бухгалтерскому учету «Учет расходов на научно-исследовательские, опытно-конструкторские и технологические работы организации» (ПБУ 17/02).

5. Положение по бухгалтерскому учету «Отчет о движении денежных средств» (ПБУ 23/2011).

Стандарты:

1. Федеральный стандарт по бухгалтерскому учету «Бухгалтерская (финансовая) отчетность» (ФСБУ 4/2023)

2. Федеральный стандарт по бухгалтерскому учету «Запасы» (ФСБУ 5/2019).

3. Федеральный стандарт по бухгалтерскому учету «Основные средства» (ФСБУ 6/2020).

4. Федеральный стандарт по бухгалтерскому учету «Документы и документооборот в бухгалтерском учете» (ФСБУ 27/2021).

5. Федеральный стандарт по бухгалтерскому учету «Нематериальные активы» (ФСБУ 14/2022).



Состав отчетности:
1. Бухгалтерский баланс
2. Отчет о финансовых результатах
3. Отчет об изменениях капитала
4. Отчет о движении денежных средств
5. Пояснение к бухгалтерскому балансу
6. Отчет о целевом использовании полученных средств
- Аудиторское заключение, подтверждающее достоверность бухгалтерской отчетности предприятия, если оно в соответствии с федеральными законами подлежит обязательному аудиту.
