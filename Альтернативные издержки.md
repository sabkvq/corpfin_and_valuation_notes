Возникают из-за:
- Ограниченности ресурсов -> необходимость выбирать
- Конкурентность -> ответственность капиталом

Рассматриваются
- Операции (Сопоставимые по уровню риска )
	- Фактические
	- Потенциально возможные
- Для операций учитываются и суммируются
	- Утраченные прибыли (полученные другими компаниями с таким же уровнем риска для капитала)
	- Фактические расходы
